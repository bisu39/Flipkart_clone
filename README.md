# Flipkart Clone

A responsive frontend clone of the Flipkart landing page, showcasing modern web development skills with HTML, CSS, and JavaScript.

## 🚀 Live Demo

Check out the live demo: [https://bisu39.github.io/Flipkart_clone/](https://bisu39.github.io/Flipkart_clone/)

## 📋 Features

- **Responsive Design**: Optimized for both desktop and mobile devices
- **Interactive Navigation**: Hamburger menu, search bar, and user login dropdown
- **Product Categories**: Hover effects for Fashion, Electronics, Home & Furniture, and Beauty sections
- **Dynamic Carousel**: Auto-cycling progress indicators for featured content
- **Modern UI**: Clean and intuitive interface mimicking Flipkart's design

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Responsive styling with media queries
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Remix Icons**: Icon library for visual elements

## 📁 Project Structure

```
Flipkart Clone/
├── index.html          # Main landing page
├── script.js           # Main JavaScript functionality
├── style-desktop.css   # Desktop styles
├── style-mobile.css    # Mobile responsive styles
└── assets/ 
    └── images/         # Images and icons
```

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bisu39/Flipkart_clone.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Flipkart_clone
   ```

3. Open `index.html` in your web browser or use a local server.

### Running Locally

You can run this project locally by:

- Double-clicking `index.html` to open in your default browser
- Using a local server like Live Server extension in VS Code

## 📱 Usage

- **Navigation**: Use the hamburger menu on mobile or explore categories on desktop
- **Categories**: Hover over category sections to see sub-options
